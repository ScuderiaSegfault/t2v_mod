# Track 2 Vehicle Module

The detailed documentation about the T2V module can be found [here](doc/).

Other repositories:

- [**Firmware**](https://github.com/ScuderiaSegfault/t2v_mod_fw) for the ESP32P4 board
- **Drivers** and utilities to interface with the T2V module over USB
- **t2v_msgs** Message type definitions for the ROS interface
- **t2v_node** ROS2 node that interfaces with the background driver
